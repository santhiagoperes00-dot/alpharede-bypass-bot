# 🚀 Alpharede Bypass Bot (Telegram + Square Cloud)

Este é um bot profissional para Telegram desenvolvido para burlar automaticamente as 4 a 5 etapas de encurtadores como `alpharede.com` e `forumdinheiro.com`.

## 🛠️ Funcionalidades
- **Bypass Automático:** Pula todas as etapas de anúncios e timers.
- **Multi-Etapas:** Lida com o fluxo de 5 páginas seguidas.
- **Pronto para Square Cloud:** Configuração otimizada.
- **Interface via Telegram:** Comandos simples.

## 📦 Como Instalar

### 1. Criar o Bot no Telegram
1. Fale com o [@BotFather](https://t.me/botfather) no Telegram.
2. Use `/newbot` e siga as instruções.
3. Copie o **API Token**.

### 2. Hospedar na Square Cloud
1. Crie um arquivo `.zip` com todos os arquivos deste repositório.
2. No painel da [Square Cloud](https://squarecloud.app/), clique em "Upload".
3. Vá em **Variáveis de Ambiente**.
4. Adicione `TELEGRAM_TOKEN` com o seu token.
5. Inicie o bot.

## 🚀 Como Usar
No Telegram, use o comando:
```
/bypass https://alpharede.com/seu-link
```
